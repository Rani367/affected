The Problem

Monorepo CI is broken.

Every team ends up writing brittle bash scripts or buying into heavyweight build frameworks.

Without affected
  • × Running all 200 tests when only 1 file changed
  • × Brittle git diff | grep scripts with no dependency awareness
  • × Nx/Bazel/Turborepo: 200-500MB of node_modules or JVM just to detect changes
  • × Hours of setup, YAML boilerplate, and framework lock-in
With affected
  • ✓ Auto-detects your ecosystem — zero configuration
  • ✓ Transitive dependency graph — knows the full blast radius
  • ✓ Single ~5MB binary. No runtime dependencies.
  • ✓ Install in 1 minute, running in your CI in 5
Before
# 47 lines of fragile bash... CHANGED=$(git diff --name-only origin/main) for pkg in packages/*; do if echo "$CHANGED" | grep -q "$pkg"; then # no transitive deps, no graph cd "$pkg" && npm test && cd .. fi done
→
After
affected test --base main # That's it. Zero config. # Auto-detects ecosystem. # Builds dependency graph. # Runs only affected tests.

How It Works

Six steps. Milliseconds.

From git diff to targeted test execution, fully automated.

Detect
Scans for marker files to identify your ecosystem
→
Resolve
Parses manifests into a dependency graph
→
Diff
Computes changed files via libgit2
→
Map
Maps each file to its owning package
→
Traverse
Reverse BFS to find all dependents
→
Execute
Runs commands for affected packages only

CI Integration

CI-native by design.

GitHub Actions — Dynamic Matrix
jobs: detect: runs-on: ubuntu-latest outputs: matrix: ${{ steps.affected.outputs.matrix }} steps: - uses: actions/checkout@v4 - uses: Rani367/setup-affected@v1 - id: affected run: affected ci --merge-base origin/main test: needs: detect strategy: matrix: ${{ fromJson(needs.detect.outputs.matrix) }} steps: - run: cargo test -p ${{ matrix.package }}

Install

Get running in 60 seconds.

Choose your preferred method. All roads lead to affected.

$ brew install Rani367/tap/affected
macOS and Linux. Installs the latest binary release.
$ uv tool install affected
Also works with pipx install affected or pip install affected.
$ cargo install affected-cli
Build from source. Requires Rust toolchain.
- uses: Rani367/setup-affected@v1 - run: affected test --merge-base origin/main
Add to any GitHub Actions workflow. Auto-downloads the right binary.

Or download a binary from GitHub Releases.
